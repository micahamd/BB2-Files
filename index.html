Repository Contents

This repository includes the following files:

  • data1.csv - Primary dataset containing participant information, conditions, and outcome scores
  • data2.csv - Secondary dataset containing trial-level data including response times and accuracy measures
  • analysis.R - R script for statistical analysis and visualization of the experimental data
  • ethics_application.txt - Ethics application form documenting the ethical approval for this study
  • index.html - This documentation page

Data Description

Dataset 1 (data1.csv)

Variable Description Type
id Unique record identifier Integer
participant Participant ID (anonymized) String
age Participant age in years Integer
condition Experimental condition (control/experimental) Categorical
score Outcome measure score Numeric
date Date of data collection Date

Dataset 2 (data2.csv)

Variable Description Type
id Unique record identifier Integer
participant Participant ID (anonymized) String
trial Trial number Integer
response_time Response time in milliseconds Numeric
accuracy Response accuracy (0=incorrect, 1=correct) Binary
notes Additional notes String

Using the Data

Requirements

To run the analysis script, you will need:

  • R (version 4.0 or higher recommended)
  • Required R packages: tidyverse, ggplot2

Running the Analysis

To reproduce the analysis:

  1. Clone or download this repository
  2. Install required R packages: install.packages(c("tidyverse", "ggplot2"))
  3. Set your working directory to the repository folder
  4. Run the analysis script: source("analysis.R")

Expected Outputs

The analysis script will generate:

  • Summary statistics for both datasets
  • Statistical test results (t-test)
  • Effect size calculations (Cohen's d)
  • Visualization: scores_by_condition.png
